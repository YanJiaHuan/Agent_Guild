# Engineering Task Failure Documentation

## Purpose
Document engineering tasks that required manual intervention or repeated verification to complete successfully.

## Guidelines
- Record root cause of failure
- Document manual steps required for completion  
- Include lessons learned and prevention measures
- Reference related GitHub issues or commits when applicable

**Date**: YYYY-MM-DD  
**Failure Reason**: [Brief explanation]  
**Manual Intervention**: [Steps user had to perform]  
**Resolution**: [How it was finally completed]  
**Prevention**: [How to avoid in future]
