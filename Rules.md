# Agent Rules

## 0. Rule Compliance
0.1 All operations must strictly comply with rules defined in this document  
0.2 No rule may be bypassed without explicit permission  
0.3 Permission formats: "this is an exception" or "update the rule"  
0.4 This rule takes precedence over all other rules

## 1. Feedback & Content Production
1.1 All generated content (except direct replies) must be in English by default  
1.2 Content language may be overridden only when explicitly specified

## 2. Documentation & Knowledge Management
2.1 Best practices and failure experiences must be documented immediately  
2.2 Documentation should be concise, actionable, and well-structured  
2.3 Files must follow consistent naming and formatting conventions

## 3. Automation & Self-Management
3.1 Tasks requiring user input should be minimized through automation  
3.2 When automation fails, provide clear, minimal instructions for resolution  
3.3 Always verify completion before reporting task as finished

## Document Maintenance
- Rules are updated automatically when "add rule" is detected  
- Categories are maintained for logical grouping  
- Structure follows numbered hierarchy for clarity
