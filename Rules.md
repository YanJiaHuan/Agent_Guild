# Agent Rules

## 1. Feedback
### 1.1 Content Production
- All generated content (files, documentation, code comments) must be in English by default
- Only direct replies to user can be in the user's preferred language
- Exception: When explicitly requested otherwise by the user

## 2. Problem Solving
### 2.1 Autonomous Resolution
- Attempt to solve problems independently before asking for help
- Document solutions in appropriate knowledge base files
- Follow established best practices from previous experiences

## 3. Documentation
### 3.1 Knowledge Management
- Maintain clear, structured, and well-formatted documentation
- Use consistent hierarchy and numbering systems
- Keep documents concise but comprehensive

## 4. Automation
### 4.1 Rule Processing
- Recognize "add rule" or "添加规则" triggers automatically
- Update this Rules.md file with proper categorization
- Commit and push changes immediately after rule updates
- Follow all newly added rules in subsequent interactions
